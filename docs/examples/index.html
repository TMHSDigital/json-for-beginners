JSON Examples

Explore a variety of JSON examples, from basic objects to complex API responses. Each example is ready to copy, validate, and use.

Basic Examples

Basic Object

A simple JSON object with common data types.

{
  "name": "John Doe",
  "age": 30,
  "email": "john@example.com",
  "isActive": true,
  "tags": ["developer", "javascript", "json"],
  "address": {
    "street": "123 Main St",
    "city": "Boston",
    "country": "USA"
  }
}

Nested Structure

A complex JSON object with nested objects and arrays.

{
  "company": {
    "name": "TechCorp",
    "founded": 2010,
    "departments": [
      {
        "name": "Engineering",
        "employees": [
          {
            "id": 1,
            "name": "Alice Smith",
            "role": "Senior Developer",
            "skills": ["JavaScript", "Python", "Docker"]
          }
        ]
      }
    ]
  }
}

API Response

A typical API response structure with metadata.

{
  "status": "success",
  "code": 200,
  "data": {
    "users": [
      {
        "id": "usr_123",
        "username": "johndoe",
        "profile": {
          "fullName": "John Doe",
          "email": "john@example.com"
        }
      }
    ],
    "pagination": {
      "total": 150,
      "page": 1,
      "perPage": 10
    }
  }
}
