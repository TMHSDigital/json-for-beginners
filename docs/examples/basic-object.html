Basic JSON Object

A basic JSON object represents a single entity with multiple properties. This example shows a book object with various attributes.

Example

{
  "title": "The Great Gatsby",
  "author": "F. Scott Fitzgerald",
  "year": 1925,
  "genre": "Novel",
  "pages": 180,
  "isAvailable": true,
  "tags": ["fiction", "classic", "american-literature"]
}

Explanation

This JSON object represents a book with the following properties:

  • title: A string containing the book's title
  • author: A string with the author's name
  • year: A number representing the publication year
  • genre: A string describing the book's genre
  • pages: A number indicating the total number of pages
  • isAvailable: A boolean indicating if the book is available
  • tags: An array of strings containing relevant tags
