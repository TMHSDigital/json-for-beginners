API Response Example

JSON is commonly used in API responses. This example shows a typical weather API response with various data points.

Example

{
  "status": "success",
  "timestamp": "2024-03-15T14:30:00Z",
  "location": {
    "city": "New York",
    "country": "USA",
    "coordinates": {
      "latitude": 40.7128,
      "longitude": -74.0060
    }
  },
  "current": {
    "temperature": 22.5,
    "feels_like": 23.0,
    "humidity": 65,
    "wind": {
      "speed": 5.2,
      "direction": "NE",
      "gust": 8.0
    },
    "conditions": "Partly Cloudy",
    "icon": "partly-cloudy-day"
  },
  "forecast": [
    {
      "date": "2024-03-16",
      "high": 24.0,
      "low": 18.0,
      "conditions": "Sunny",
      "precipitation_chance": 10
    },
    {
      "date": "2024-03-17",
      "high": 23.0,
      "low": 17.0,
      "conditions": "Cloudy",
      "precipitation_chance": 30
    }
  ],
  "alerts": [
    {
      "type": "wind",
      "severity": "moderate",
      "message": "Wind gusts up to 20 mph expected"
    }
  ]
}

Explanation

This JSON structure represents a weather API response with several key components:

  • status: Indicates if the API request was successful
  • timestamp: When the data was retrieved
  • location: Information about the location
    • city and country: Location identifiers
    • coordinates: Geographic coordinates
  • current: Current weather conditions
    • Temperature, humidity, and feels-like temperature
    • wind: Wind speed, direction, and gusts
    • conditions: Weather description
    • icon: Weather icon identifier
  • forecast: Array of future weather predictions
  • alerts: Array of weather warnings or notices
