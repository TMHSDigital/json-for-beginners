Nested JSON Structure

JSON objects can contain other objects and arrays, creating complex nested structures. This example shows a shopping cart with nested items.

Example

{
  "cartId": "CART123",
  "customer": {
    "name": "John Doe",
    "email": "john@example.com",
    "address": {
      "street": "123 Main St",
      "city": "New York",
      "state": "NY",
      "zip": "10001"
    }
  },
  "items": [
    {
      "productId": "P001",
      "name": "Laptop",
      "price": 999.99,
      "quantity": 1,
      "options": {
        "color": "Silver",
        "warranty": "2 years"
      }
    },
    {
      "productId": "P002",
      "name": "Mouse",
      "price": 29.99,
      "quantity": 2,
      "options": {
        "color": "Black",
        "wireless": true
      }
    }
  ],
  "total": 1059.97,
  "currency": "USD"
}

Explanation

This JSON structure represents a shopping cart with nested objects and arrays:

  • cartId: A unique identifier for the cart
  • customer: An object containing customer information
    • name and email: Basic customer details
    • address: A nested object with address information
  • items: An array of product objects, each containing:
    • Basic product information (ID, name, price, quantity)
    • options: A nested object with product-specific options
  • total: The total cost of all items
  • currency: The currency used for prices
